# Julsignals

Proyecto limpio de Next.js para desplegar en Vercel.

## Estructura clave
- `app/page.js`
- `app/layout.js`
- `app/globals.css`
- `app/api/analyze/route.js`
- `components/SignalCard.js`
- `lib/analysis.js`

## Despliegue
1. Sube esta carpeta a un repo nuevo de GitHub.
2. En Vercel, importa ese repo.
3. Deploy.
